# 🤖 AI-Native Product Workflow

I work directly from the **codebase**, using AI tools to reduce the traditional gap between product and engineering.

My workflow typically includes:

• Exploring systems directly from the **source code**  
• Using **Cursor, Windsurf, and Claude** to analyze repositories  
• Prototyping product ideas using **BMAD and AI tools**  
• Validating ideas with **users before writing user stories**

I believe the future of product management is **closer to the codebase and powered by AI**.
